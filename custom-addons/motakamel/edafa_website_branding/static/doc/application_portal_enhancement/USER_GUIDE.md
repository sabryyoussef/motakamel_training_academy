## ‚ùì Frequently Asked Questions

### General Questions

**Q: How long does the application take?**  
A: Approximately 10-15 minutes if you have all information ready.

**Q: Can I save my progress and come back later?**  
A: Yes! The form auto-saves every 30 seconds. Your progress is saved in your browser. Just return to the same browser and click "Apply Now" again.

**Q: What if I close my browser accidentally?**  
A: Your draft is saved! When you return, you'll be asked if you want to continue from where you left off.

**Q: Can I submit multiple applications?**  
A: Yes, but you'll receive a warning if you use the same email. You may apply for different programs/courses.

### Technical Questions

**Q: Which browsers are supported?**  
A: Chrome, Firefox, Safari, and Edge (latest versions). Mobile browsers also work.

**Q: Do I need to create an account?**  
A: No! You can apply without creating an account. However, creating an account lets you track applications easily.

**Q: Is my information secure?**  
A: Yes. We use HTTPS encryption and follow data protection regulations.

**Q: Can I edit my application after submitting?**  
A: No, but you can contact admissions to request changes. Email: support@edafa.org

### Application Questions

**Q: What if I don't know my course yet?**  
A: You can leave course/program blank. Admissions team will help you choose.

**Q: Is previous education required?**  
A: No, it's optional. Only needed for transfer students or scholarship applicants.

**Q: Why do you need family income?**  
A: Only for scholarship eligibility assessment. It's completely optional and confidential.

**Q: What photo should I upload?**  
A: Passport-sized photo with clear face. JPG, PNG, or GIF format, max 5MB. Optional but recommended.

### After Submission

**Q: When will I hear back?**  
A: Within 3-5 business days. Check your email regularly.

**Q: What happens after submission?**  
A: You'll receive:
1. Immediate confirmation email
2. Application number
3. Next steps instructions
4. Updates as your application is reviewed

**Q: Can I check my status?**  
A: Yes! Use "Check Status" menu or log in to your portal account.

**Q: I didn't receive confirmation email. What should I do?**  
A: 
1. Check spam/junk folder
2. Wait 15 minutes (may be delayed)
3. Use "Check Status" with your application number
4. Contact support if still not received

## üêõ Troubleshooting

### Form Issues

**Problem: "Next" button doesn't work**

**Solutions:**
1. Make sure required fields (marked with *) are filled
2. Check for red borders indicating errors
3. Fix any validation errors shown
4. Try a different browser
5. Clear browser cache (Ctrl+Shift+R)

---

**Problem: I can't upload my photo**

**Solutions:**
1. Check file size (must be under 5MB)
2. Use supported formats: JPG, PNG, GIF
3. Try a different photo
4. Skip photo upload (it's optional)

---

**Problem: State dropdown is empty**

**Solutions:**
1. Select a country first
2. Wait 2-3 seconds for states to load
3. Check your internet connection
4. Refresh the page

---

**Problem: Email shows warning "already has application"**

**Explanation:** This email was used for a previous application. This is just a warning - you CAN still proceed if needed (e.g., applying for a different program).

---

**Problem: Form data disappeared**

**Solutions:**
1. Check if draft recovery prompt appeared
2. Look in your browser's localStorage
3. Don't use incognito/private mode (doesn't save drafts)
4. Use the same browser you started with

---

### Validation Errors

**"Please enter a valid email address"**
- Email must be in format: name@domain.com
- No spaces allowed
- Must have @ symbol and domain

**"Date of birth must be in the past"**
- Can't select today or future dates
- Must be at least 16 years ago

**"Phone number must be at least 10 digits"**
- Enter full phone number with country code
- Format: +1-555-555-5555 or similar

**"This field is required"**
- Field marked with red asterisk (*) must be filled
- Can't proceed to next step without it

---

### Technical Issues

**JavaScript not working**
1. Enable JavaScript in browser settings
2. Disable ad blockers (may block scripts)
3. Try a different browser
4. Clear cache and cookies

**Page not loading**
1. Check internet connection
2. Refresh page (F5 or Ctrl+R)
3. Clear browser cache
4. Try incognito/private mode

**Draft not saving**
1. Enable cookies in browser
2. Don't use incognito mode
3. Check localStorage is enabled
4. Try a different browser
